## 有点长的前言

最近一段时间回坑了迦勒底，主要原因是听说十周年直接送 1000 圣晶石，同时还有超强力的周年从者奥尔加玛丽所长。我虽然不是什么老资历月丑，但是好歹也是空境活动入坑的，对于这个游戏也是有点感情的。另一方面看到了第二步完结的消息，虽然很多人对这个结局不满意，但是我倒是无所谓。对我来说我作为迦勒底人类最后的御主，不管结局怎么说，至少也得我自己亲自打完才能评价一下吧，甚至所长的手我还没抓住呢，至少也得让我抓一下吧。

当然，还有一个非常重要的原因，就是这篇文章要介绍的了，一个功能非常强大的工具 ──「BB Channel」，以下为了省事就直接称为 BBC 好了。其实之前为什么我玩 FGO 玩腻了呢，一方面是活动剧情有点懒得看了，另一方面就是 FGO 刷起来确实有点繁琐。虽然之前我已经再用模拟器的录制功能来减轻操作，模拟器只能录制固定的操作，每刷一次本还得手动选择助战，吃苹果等等，实际上只能说有一点帮助。所以我一直在想，能不能编写一个工具，通过图像识别的方式能够自动选择助战、吃苹果、自动选择指令卡等等，这样弄一个全自动化的助手，那样岂不是强无敌？我甚至还在我的 GitHub 上创建了这么一个仓库，虽然什么都没有写。

不过现在有了 BBC，情况就不一样了。虽然离我想象的全自动刷本还有一定距离，但是 BBC 已经实现了最重要的自动助战和苹果功能，也就是说我们现在可以完全自动的刷本了。只需要在 BBC 中设置好自己的队伍，录制好操作之后，剩下的就可以交给 BBC 了。他会自动按照你的指令每回合执行对应操作，在战斗完成后自动选择助战，在体力耗尽后自动吃苹果。虽然和我之前的模拟器录制相比只是多了自动助战和吃苹果两项，但是体验却是质的飞跃，因为刷本的时候实现了完全的无人值守，爱干嘛干嘛，再也不用每个一两分钟盯一下。而且我回来的时候刚刚好是月批公主的活动，还有刷柱子战的环节，使用 BBC 简直就是爽到飞起。以前柱子战模拟器录制也就刷个几十次。这个活动轻轻松松 500+,量子也完全溢出。现在我还有 700+ 个苹果，但是已经产生了体力焦虑，因为用 BBC 的话，下次碰到高收益的柱子战或者无限池，这 700 苹果挂个一天一夜我看也就用完了。

不过说起月批活动，又成了我的一个伤心事，本来回坑高高兴兴的，把以前的石头已领攒了差不多 800+石头，查了一下这个月批公主也是强度很高的从者，我寻思那就抽一下吧，结果没想到 800+的石头只有一宝公主，四星倒是歪了不少，着实把我整无语了。当然最坑的还是我都忘了保底这回事，刷柱子战的时候没注意后排羁绊，假如仔细选择一下后排羁绊的话，应该能多刷两个十绊，这不就正好把保底的石头凑够了吗，至少二宝是稳了。

沉船之后，我对未来的所长也没有太大期望了，本来想着这大半年把活动做的差不多，加上周年庆送的 1000 石，追个五宝所长也稳了。剩下的刚好还有个全活动池子，在追个二宝或者三宝的水伊吹，我 FGO 岂不是大成了。但是现在脸这么黑，再来一次千石一宝，那我又得退坑了。不过到时候反正第二部也完结了，也算了却自己一个念想吧。反正到时候再说吧，说不定千石五宝所长呢。

## BBC 使用

好了，废话说的有点多了，开始今天的正题吧。

在此感谢 [@小阮绵绵咩](https://space.bilibili.com/36142005) 大佬开发的 BBC 工具。

### 下载说明

BBC 的作者在 B 站，工具没有开源，也只能通过作者的网盘下载。软件的使用说明可以通过作者的专栏来查看：

- [作者专栏列表](https://www.bilibili.com/read/readlist/rl474502)

#### 安装包下载

软件下载分为**基础包**和**更新包**两个部分，两个都需要下载，更新包需要覆盖到基础包上。

**Google 网盘链接（推荐）：**

- **3.62 基础包**：<https://drive.google.com/file/d/1pAvP1cO5Y58oR6XUB8tyNYxnEKZOY37A/view?usp=sharing>
- **3.79 更新包**：<https://drive.google.com/file/d/1GnPtHCx7GrZF91o_2PmrTaH3safFUwtZ/view?usp=sharing>

> **注意**：将来作者发布新版的时候这些链接可能过时，建议到作者 B 站动态或专栏查找最新版本。

#### 安装步骤

1. 解压基础包到任意目录
2. 双击其中的 `启动.cmd` 打开 BBC
3. 点击菜单上的「更新咨询与支持」
4. 选择右下角的「安装更新包」
5. 选择刚刚下载的更新包（**不需要解压**）
6. 点击更新，等待完成
7. 重新打开软件即可

> 💡 **提示**：基础包的压缩包可以先留着，以后万一更新出问题可以用这个再重来。

![更新bbc](https://i0.hdslb.com/bfs/new_dyn/48a5e1f87b5eb1aff8f0439c42b0aac51516948.png)

### 连接模拟器

模拟器其实用什么都可以，但是**强烈推荐使用 MuMu 模拟器**。BBC 会通过截图分析图像，只要能 ADB 或者抓取窗口内容都可以连接，对应 Win 窗口连接和 ADB 端口连接等几种方式。

#### 推荐：MuMu 高速连接

经过实测，MuMu 高速连接是最稳定的连接方式：

1. **不需要关闭应用保活设置**，可以后台挂 FGO，前台玩别的游戏
2. 直接调用 MuMu 模拟器的 API，功能更强大
3. 性能最好，不会出现点击无反应的情况

**连接步骤：**

1. 点击 BBC 菜单中的「连接」→「MuMu 高速连接」
2. 输入 MuMu 模拟器的安装路径（默认：`C:\Program Files\NetEase\MuMu`）
3. 点击「应用」
4. 成功后会列出设备名称
5. 点击下方的「连接」即可

![连接模拟器](https://i0.hdslb.com/bfs/new_dyn/ef2d9dd2e1c578c65fe4c259144cd9b71516948.png@1192w.avif)

**测试连接：**

连接成功后，点击 BBC 菜单「连接」→「测试连接状态」，能看到 FGO 游戏内容就说明成功了。

### 队伍配置

接下来就是队伍配置了，从这部分开始就开始麻烦起来了。

#### 配置从者

1. 点击放大镜可以筛选查找从者
2. **从者的顺序必须和你游戏中实际队伍完全一致，包括助战**
3. 游戏中是第几位，BBC 中就得是第几位
4. 用到几个从者就配置几个，不用的位置留空
5. 别忘了选择御主服

> 例如：如果队伍只需要前三个从者，那么后面三个位置就留空，实际游戏中放什么羁绊位都可以。

### 操作设置

这是 BBC 使用的重点，设置操作序列。

#### 操作序列设置步骤

1. 点击「操作序列」按钮，展开操作设置
2. 点击「第一面技能」，按钮从**橙色变为黄色**
3. 此时从者技能会显示出来
4. 像正常刷本一样配置操作，释放技能，放完技能再点击从者上方的「本回合放宝具」按钮
5. 点击「保存」完成第一面设置
6. 点击「第二面技能」，同样变为黄色后设置第二面
7. 依此类推设置第三面

![操作设置](https://i0.hdslb.com/bfs/new_dyn/e550e617894bc5a477ec30713d3e30e41516948.png@1192w.avif)

![操作设置2](https://i0.hdslb.com/bfs/new_dyn/a9669ea248fbd76c377f1595c465ce101516948.png@1192w.avif)

#### 特殊情况：柱子战/一面本

如果是柱子战之类只有一面或二面的：

- 第三面设置直接点击「保存」
- 会询问是否返回，选择「否」就是跳过这个回合

> 💡 看不明白多试两次就懂了，反正设置错了可以重置重来。

### 刷本设置

#### 设置刷本次数

点击「刷本次数」按钮可以设置：

- **刷本次数**：要刷多少次
- **苹果类型**：要吃哪种苹果（金苹果/银苹果/铜苹果/蓝苹果）
- **刷本类型**：一般选择「连续出击」即可

#### 其他设置选项

点击「设置」菜单可以配置更多选项，看情况选择：

| 选项         | 说明                                       |
| ------------ | ------------------------------------------ |
| 满破礼装     | 助战是否要求满破礼装                       |
| 宝具等级     | 借助战时对宝具等级的要求                   |
| 从者等级     | 助战从者等级要求                           |
| 允许其他苹果 | 是否允许吃其他类型的苹果                   |
| 满羁绊停止   | 满羁绊后是否自动停止                       |
| 自动加好友   | 是否自动发送好友申请                       |
| 首次调速     | **首次使用建议关闭**，否则可能导致乱点技能 |

### 助战设置（礼装作成）

这是 BBC 最复杂但也是最强大的功能之一，可以精确筛选助战礼装。

#### 礼装作成步骤

**第一步：游戏内准备**

在游戏内将界面选到**助战选择界面**，同时显示出你想要选择的助战从者和礼装。

![助战选择](https://i0.hdslb.com/bfs/new_dyn/1b9286e3bc3f4ed3e1b90f2be66ffa1a1516948.png@1192w.avif)

**第二步：BBC 中展开礼装选择**

1. 点击 BBC 中的助战空白处，展开「礼装选择」
2. 选择「礼装作成」

![选择礼装做成](https://i0.hdslb.com/bfs/new_dyn/4624762f99919234343587053eb17efc1516948.png@1192w.avif)

**第三步：截图**

- 点击「获取满破礼装图」或「获取未满破礼装图」按钮
- 这两个按钮其实就是截图按钮，会截取礼装部分的截图

![截取礼装图](https://i0.hdslb.com/bfs/new_dyn/e59d08daf04a8fd3d14ccf5d155caa8b1516948.png@1192w.avif)

**第四步：选择截图**

- 点击「选择满破礼装图」或「选择未满破礼装图」按钮
- 选择刚刚截下来的图
- 列表中会显示出一些礼装截图，选择准确的那一张

![选择礼装图](https://i0.hdslb.com/bfs/new_dyn/ed7c7c0cb3ce39978c03ef52cab5f8411516948.png@1192w.avif)

**第五步：保存礼装**

1. 在上方文本框输入名称（随便起，如「倍卡」「活动礼装」等，就是文件名）
2. 点击「礼装作成」按钮保存

![测试礼装图](https://i0.hdslb.com/bfs/new_dyn/23bcf19fea7a8e3bfcc22f1030cf18431516948.png@1192w.avif)

**第六步：测试与添加**

1. 点击「测试礼装识别效果」，找到刚刚保存的礼装图
2. 确认能成功识别到
3. 点击「加入主界面选择器」，选择刚刚的礼装

完成以上步骤后，助战从者就可以选择这个礼装，BBC 会据此来筛选助战。

### 开始战斗

全部设置好了之后：

1. 在游戏里选好助战，进入战斗界面
2. 在 BBC 中点击「开始」
3. 享受全自动打本的乐趣！

BBC 会按照你的设置：

- 每回合自动释放技能
- 自动选择宝具
- 战斗结束后自动选择助战
- 体力耗尽后自动吃苹果
- 继续下一轮战斗

以后 FGO 不再是一个磨磨叽叽的时间消耗杀手，你从此变成了时间的主人！更别说以后什么柱子战之类的，研究好配队羁绊礼装之后，一口气直接刷满羁绊，你就是迦勒底的皇帝！

### 常见问题

#### 1. 识别不准确

需要在游戏里**关闭战斗中语音字幕**，这个可能影响 BBC 的识别效果。

![战斗语音字幕](https://i0.hdslb.com/bfs/new_dyn/a75d86d409da548752d2f4c048a451e21516948.png@1192w.avif)

#### 2. 模拟器连接成功但是点击开始没反应

测试的时候可以看到游戏画面，但是无法操作。

**解决办法**：更换连接类型，推荐使用 **MuMu 高速连接**，效果最好。

#### 3. 开始战斗之后 BBC 乱点技能

这是常见错误，第一次用的时候很容易遇到。

**解决办法**：在设置里**关闭「开启首次调速」**选项，如果不行的话在试试关闭**不加速技能动画**选项。

![关闭首次调速](https://i0.hdslb.com/bfs/new_dyn/87496cd5848b2f525d2172e49ef1210f1516948.png@1192w.avif)

## 总结

BBC 的出现可以说是解放了大部分玩家的双手，以后刷本再也不用手动操作了，极大简化操作。如果我退坑那会就有了这个工具的话，我肯定就不会退坑了。

当然，BBC 离我理想的 FGO 助手还有一段距离，因为我想象的功能实在是太多了。我想着把所有 FGO 数据都包括进来，实现游戏内和助手内数据的同步，然后实现一些高级功能，比如全自动推图，自动根据仓库计算合适阵容配置等等，这些理论上都是可以做到的。

当然我想得太多太复杂，所以到头来一行代码也没有实现，反倒是人家脚踏实地，虽然软件界面丑了点，但是却把核心功能都实现了出来。这一点值得我好好学习学习。
