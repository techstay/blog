# my-static-blog [![Build Status](https://travis-ci.com/techstay/my-static-blog.svg?branch=master)](https://travis-ci.com/techstay/my-static-blog)

我的基于Hexo的静态博客

## 创建新文章

```
$ hexo new [layout] <标题>
```

## 本地运行

```
$ hexo server
```

## 生成博客

```
$ hexo generate
```

## 部署博客

```
$ hexo deploy
```

当然如果配置了Travis CI的话就更简单了，直接推代码就可以了。
